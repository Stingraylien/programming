## 1. 系統概述

本系統接收課程教材（PDF、PPTX、DOCX、TXT），自動產出以下五種成果：

| # | 檔案類型 | 說明 |
|---|---------|------|
| 1 | `.txt` | 含課程名稱、課程目標、課程大綱的純文字摘要 |
| 2 | `.pptx` | 課程重點摘要投影片（深藍主題，16:9） |
| 3 | `.mp4` | 以摘要投影片為基底，配合 TTS 語音旁白的 H.264 影片 |
| 4 | `.xlsx` | 符合旭聯 LMS 格式的考題題庫（最低 20 題） |
| 5 | `.jpg` | 依課程名稱與內容自動生成的封面圖片（1280×720） |

## 2. 環境需求與安裝

### 2.1 系統需求

- Python 3.9 以上
- macOS / Windows / Linux 均支援
- 需有網路連線（Edge TTS 語音生成需連線至 Microsoft 雲端服務）
- 需安裝 [FFmpeg](https://ffmpeg.org/download.html)（MP4 影片輸出必要）

### 2.2 安裝 FFmpeg

**macOS（使用 Homebrew）：**
```bash
brew install ffmpeg
```

**Windows：**
1. 前往 [https://ffmpeg.org/download.html](https://ffmpeg.org/download.html) 下載
2. 解壓後將 `bin/` 資料夾路徑加入系統 `PATH` 環境變數

**Ubuntu / Debian：**
```bash
sudo apt install ffmpeg
```

### 2.3 安裝 Python 套件

在專案根目錄執行：
```bash
pip install -r requirements.txt
```

主要依賴套件包含：`flask`、`python-pptx`、`edge-tts`、`moviepy`、`Pillow`、`openpyxl`、`PyPDF2`、`python-docx`。

## 3. 啟動伺服器

在專案根目錄執行：

```bash
python api/main.py
```

伺服器預設啟動於 `http://localhost:8000`。

確認伺服器正常運作：
```bash
curl http://localhost:8000/api/health
```

預期回應：
```json
{"status": "healthy", "timestamp": "2026-05-13T10:00:00.000000"}
```

## 4. API 參數說明

### 端點

```
POST http://localhost:8000/api/generate-question-bank
Content-Type: multipart/form-data
```

### 參數一覽

| 參數名稱 | 類型 | 必填 | 說明 |
|---------|------|------|------|
| `materials` | 檔案（可多個） | 是 | 課程教材，支援 `.pdf`、`.pptx`、`.docx`、`.txt`、`.mp4` |
| `course_name` | 字串 | 是 | 課程名稱，作為所有產出檔案的命名基礎 |
| `question_types` | 字串（可多個） | 是 | 題型，可同時指定多種（見下表） |
| `difficulty` | 字串 | 否 | 難易度：`easy`、`medium`（預設）、`hard` |
| `language` | 字串 | 否 | 輸出語系（預設 `zh-TW`） |
| `target_language` | 字串 | 否 | 語音旁白語系（預設 `zh-TW`） |
| `source_language` | 字串 | 否 | 來源教材語系（預設 `auto`，即不翻譯） |

### 支援題型

| 值 | 說明 |
|----|------|
| `true_false` | 是非題 |
| `single_choice` | 單選題 |
| `multiple_choice` | 複選題 |
| `fill_in_blank` | 填空題 |
| `essay` | 問答題 |

### 支援語系

| 值 | 語言 |
|----|------|
| `zh-TW` | 繁體中文（預設） |
| `zh-CN` | 簡體中文 |
| `en-US` | 英文 |
| `ja-JP` | 日文 |
| `es-ES` | 西班牙文 |
| `cs-CZ` | 捷克文 |
| `th-TH` | 泰文 |

## 5. 使用方式（cURL）

### 5.1 基本範例（繁中，單一教材）

```bash
curl -X POST http://localhost:8000/api/generate-question-bank \
  -F "materials=@課程教材.pdf" \
  -F "course_name=資訊安全基礎課程" \
  -F "question_types=true_false" \
  -F "question_types=single_choice" \
  -F "question_types=multiple_choice" \
  -F "difficulty=medium" \
  -F "language=zh-TW"
```

### 5.2 多檔案上傳

```bash
curl -X POST http://localhost:8000/api/generate-question-bank \
  -F "materials=@chapter1.pdf" \
  -F "materials=@chapter2.pptx" \
  -F "course_name=職業安全衛生訓練" \
  -F "question_types=true_false" \
  -F "question_types=single_choice" \
  -F "difficulty=hard"
```

### 5.3 成功回應格式

```json
{
  "status": "success",
  "message": "Successfully generated 25 questions",
  "questions_count": 25,
  "files": {
    "txt_url": "/api/download/職業安全衛生訓練.txt",
    "ppt_url": "/api/download/職業安全衛生訓練.pptx",
    "video_url": "/api/download/職業安全衛生訓練.mp4",
    "excel_url": "/api/download/職業安全衛生訓練.xlsx",
    "cover_url": "/api/download/職業安全衛生訓練.jpg"
  },
  "timestamp": "2026-05-13T10:05:32.000000"
}
```

### 5.4 下載產出檔案

取得回應中的 URL 後，使用下列指令下載（以 Excel 為例）：

```bash
curl -O http://localhost:8000/api/download/職業安全衛生訓練.xlsx
```

## 6. 使用方式（Python）

```python
import requests

url = "http://localhost:8000/api/generate-question-bank"

files = [
    ("materials", ("課程教材.pdf", open("課程教材.pdf", "rb"), "application/pdf"))
]

data = {
    "course_name": "資訊安全基礎課程",
    "question_types": ["true_false", "single_choice", "multiple_choice"],
    "difficulty": "medium",
    "language": "zh-TW",
    "target_language": "zh-TW"
}

response = requests.post(url, files=files, data=data)
result = response.json()

if result["status"] == "success":
    print(f"產出題數：{result['questions_count']}")
    for key, path in result["files"].items():
        print(f"{key}: http://localhost:8000{path}")
else:
    print(f"錯誤：{result.get('error')}")
```

## 7. 產出檔案說明

所有產出檔案儲存於專案的 `outputs/` 資料夾，檔名統一以 `course_name` 命名。

### 7.1 純文字摘要（`.txt`）

格式範例：
```
==================================================
課程名稱：資訊安全基礎課程
==================================================

【課程目標】
  1. 了解資訊安全的基本概念
  2. 掌握常見威脅與防護措施
  ...

【課程大綱】
  1. 資訊安全概論
  2. 網路攻擊類型
  ...

--------------------------------------------------
產出時間：2026-05-13 10:05
```

### 7.2 摘要投影片（`.pptx`）

- **尺寸**：16:9（寬 33.87cm × 高 19.05cm）
- **主題**：深藍底色標題頁 + 淺藍內容頁
- **第 1 頁**：課程名稱 + 「課程重點摘要」副標
- **後續各頁**：每份教材一頁，含章節名稱、學習目標（粗體條列）、段落摘要（縮排條列）
- 可直接於 PowerPoint / Keynote 開啟後進行二次編輯

### 7.3 課程影片（`.mp4`）

- **解析度**：1280×720（HD）
- **編碼**：H.264 + AAC，支援 Windows 媒體播放器、QuickTime、手機等各平台播放
- **內容**：封面投影片（含開場語音）+ 每個章節的重點投影片（含 TTS 語音旁白）
- **語音**：使用 Microsoft Edge TTS，預設繁中女聲（`zh-TW-HsiaoChenNeural`）

### 7.4 旭聯考題題庫（`.xlsx`）

- 符合旭聯 LMS 平台匯入規範
- 最低產出 20 題
- 欄位包含：題型、題目、選項、配分、難度、答案、解析、禁止亂序等

### 7.5 封面圖片（`.jpg`）

- **尺寸**：1280×720
- **內容**：深藍漸層背景、課程名稱主標題、自動擷取的關鍵主題標籤

## 8. 常見問題與注意事項

### Q：上傳 MP4 影片後，考題內容很空洞？
A：MP4 影片的語音轉文字功能尚未實作，目前 MP4 輸入僅提供佔位處理。建議搭配 PDF 或 PPTX 教材使用。

### Q：影片產出時間很長？
A：MP4 生成需要呼叫雲端 TTS 服務並以 FFmpeg 編碼，視教材章節數量，通常需要 30 秒至數分鐘不等。請耐心等待，不要中斷請求。

### Q：語音聽起來不自然？
A：旁白文字由規則組合生成，非 AI 撰寫，建議手動調整 `.txt` 摘要後再另行製作旁白。

### Q：產出的 PPTX 字型跑掉？
A：PPTX 使用系統字型，在不同電腦開啟時若缺少對應字型可能發生替換。建議在有安裝中文字型的電腦上開啟，或將 PPTX 匯出為 PDF 再分享。

### Q：Edge TTS 呼叫失敗？
A：請確認網路連線正常。Edge TTS 需連線至 `speech.platform.bing.com`，若公司網路有防火牆限制，請洽 IT 人員開通。

### Q：上傳檔案大小限制？
A：預設上限為 100 MB（可於 `config.py` 的 `MAX_CONTENT_LENGTH` 調整）。

### Q：如何取得系統支援的所有參數選項？
A：
```bash
curl http://localhost:8000/api/config
```
