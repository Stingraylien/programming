# PDF to Markdown Converter

這是一個由 Python 撰寫、具備圖形化介面（GUI）的小型實用工具。主要功能是能夠一鍵將選定資料夾中（包含所有的子資料夾）的 `.pdf` 檔案，全部自動轉換成易於閱讀與後續處理的 Markdown (`.md`) 格式。

## 解決的痛點
在處理大量 PDF 文件（例如報告、論文、手冊等）時，如果需要提取純文字、標題架構或是要餵給 Large Language Model (LLM) 處理，傳統的 PDF 非常難以操作。本工具透過自動化的方式，省去了手動轉換、複製貼上的時間，並提供最親民的圖形化介面，讓非技術人員也能輕鬆操作。

## 核心功能總覽
- 🖱️ **直覺的圖形介面**：使用 PySide6 建立的優雅介面，免去輸入複雜指令的困擾。
- 🔍 **遞迴搜尋檔案**：只需選擇最上層的資料夾，系統會自動翻找所有層級的子目錄。
- 📊 **即時進度追蹤**：支援進度條、檔案數統計（總數、已完成、剩餘）以及滾動式文字日誌。
- ⚡ **高效能核心引擎**：採用 `pymupdf4llm`，提供兼具速度與排版保留的優秀轉換品質。

## 快速啟動指令
請在您的終端機（Terminal）中輸入以下指令啟動：

```bash
python pdf_to_md_converter.py
```
