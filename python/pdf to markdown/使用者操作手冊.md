# PDF 轉 Markdown 小工具 - 使用者操作手冊

歡迎使用「PDF 轉 Markdown 小工具」！這份手冊將教您如何輕鬆地將大量的 PDF 檔案轉換成文字編輯友善的 Markdown 格式。完全不需要懂任何程式碼，只要會用滑鼠點擊即可完成！

---

## 🚀 步驟式教學

### 第一步：啟動程式
請在終端機中執行程式（或者由系統管理員為您建立捷徑點開）。啟動後，您會看到一個名為「PDF to Markdown Converter」的視窗。

### 第二步：點擊「Select Folder」按鈕
在視窗正上方有一顆藍色的大按鈕 `Select Folder`。點下去之後，系統會彈出一個選擇資料夾的視窗。

### 第三步：選取您要處理的資料夾
在彈出的視窗中，找到您存放 PDF 檔案的資料夾並選取它。
> **💡 小提示**：程式會自動幫您檢查這個資料夾**裡面所有的子資料夾**，所以您只要選擇最外層的資料夾就可以了！

### 第四步：喝杯咖啡，等待轉換
選好資料夾後，程式就會立刻開始工作。您會在畫面上看到：
- **進度條**：綠色長條圖會隨著轉換進度不斷推進。
- **統計數據**：清楚告訴您「找到了幾個檔案 (Total)」、「完成了幾個 (Completed)」、「還剩幾個 (Remaining)」。
- **日誌畫面**：畫面的文字框會跳出目前正在處理哪個檔案，以及處理的結果。

### 第五步：轉換完成！
當所有的檔案都處理完畢後，畫面會跳出一個提示視窗 `Conversion completed!`。這時您就可以回到原本存放 PDF 的資料夾，您會發現每一個 PDF 檔案旁邊，都多出了一個同名但副檔名為 `.md` 的檔案囉！

---

## ⚠️ 可能遇到的錯誤與警告

- **錯誤：`Error processing XXX.pdf: ...`**
  如果 PDF 檔案有密碼保護、檔案毀損，或是排版過於特異導致轉換核心無法讀取，日誌視窗中會跳出此行錯誤。程式會**自動略過這個檔案並繼續處理下一個**，所以不用擔心整個當機！

- **警告：`Currently processing files. Please wait.`**
  如果您在程式還沒跑完的時候，又按了一次「Select Folder」按鈕，系統會跳出這個警告。這是為了保護程式不被打斷，請耐心等它跑完喔！
