# 派遣人員匯入 e-learning 系統轉換程式

## 簡介
本專案提供一個圖形化介面 (GUI) 工具，用於將 HR 或相關單位的「派遣人員名單 (Excel 格式)」轉換為「e-learning 系統」可接受的批次匯入格式 (CSV)。支援「到職人員」與「離職人員」的針對性資料轉換，簡化人工作業流程。

## 檔案結構
- `通用_派遣人員匯入elearning.py`: 主程式 (跨平台通用版本)
- `windows_派遣人員匯入elearning.py`: Windows 專用版本
- `程式說明.md`: 核心業務邏輯與欄位對應規則說明
- `使用者操作手冊.md`: 針對終端使用者的操作步驟說明
- `開發環境建置指南.md`: 針對開發人員的環境設定與打包教學

## 快速執行
請確保已安裝 Python 環境與相關相依套件：
```bash
python 通用_派遣人員匯入elearning.py
```

## 功能特色
- 提供直覺的 GUI 按鈕介面。
- 支援讀取 `.xlsx`, `.xls`, `.csv` 等常見格式。
- 自動化欄位轉換（包含性別代碼轉譯、密碼預設邏輯等）。
- 自動忽略表頭，精準抓取資料源。
