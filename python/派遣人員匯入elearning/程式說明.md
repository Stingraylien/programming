


# 本程式會應用到 excel 相關函數
#本程式會應用到 windows 或是 mac OS 的相關檔案輸入輸出函數
#主選單
使用 GUI 介面按鈕讓使用者選取 "[到職匯入] [離職匯入] [離開]"

若 選擇 [到職匯入] 則

    使用系統 GUI 介面 來選擇檔案 要求使用者輸入一個 excel 檔案 檔案格式可接受 .xlsx .xls .csv
    Input_file = filedialog.askopenfilename(
        title="選擇 Excel 檔案",
        filetypes=[("Excel files", ".xlsx;*.xls")]
    )
    Input_file 的第四列是欄位名稱 第五列開始是資料

    # 欄位名稱如下
        工號	as text
        姓名	as text
        到職日	as (date 轉為 text) 
        性別 	as text
        出生日	as (date 轉為 text) 
        中心級名	as text
        處級名	as text
        部級名	as text
        單位名	as text
        聘僱別	as text
    自第五列開始讀入資料 一直到最後一列

    # 建立一個新的 CSV 檔案
    創立一個新的 csv 檔案 檔名為  today() +time() + "_到職人員.csv"
    第一列是欄位名稱
    # 欄位名稱如下
        EmployeeNumber	as text
        Account	as text
        FirstName	as text
        LastName	as text
        OnboardDate	as (date 轉為 text yyyy/mm/dd) 
        Gender(female=0:male=1)	as text
        Birthday	as (date 轉為 text yyyy/mm/dd) 
        Email	as text
        GradeName	as text
        DutyName	as text
        Title	as text
        Groups(Separate with a comma)	as text
        Company	as text
        OfficePhone	as text
        MobilePhone	as text
        Password	as text
        IsDisabled(no=0:yes=1)	as text
        NTAccount	as text
        InductionDate	as (date 轉為 text yyyy/mm/dd)
        #之後寫入檔案
        # 欄位名稱
        EmployeeNumber	:= Input_file 的 [工號]
        Account	:= Input_file 的 [工號]
        FirstName	:= Input_file 的 [姓名]
        LastName	:= ""
        OnboardDate	:= Input_file 的 [到職日]
        Gender(female=0:male=1) := if(Input_file 的 [性別]="女",0,1)
        Birthday	:= Input_file 的 [出生日]
        Email	:= "@"
        GradeName	:=""
        DutyName	:=""
        Title	:=""
        Groups(Separate with a comma)	:=""
        Company	:=""
        OfficePhone	:=""
        MobilePhone	:=""
        Password	:= Input_file 的 [工號]
        IsDisabled(no=0:yes=1)	:= 0
        NTAccount	:= ""
        InductionDate	:= Input_file 的 [到職日]  
    寫入檔案 
    回到 Line 7
