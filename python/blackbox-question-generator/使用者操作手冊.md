# 使用者操作手冊 (API 呼叫指南)

## 系統需求
本系統作為一個後端 API 服務運行。使用者（或前端系統/Agent）需透過 HTTP POST 請求來使用本服務。

## 操作步驟 (API 測試)

### 1. 確認服務已啟動
請確認伺服器已啟動，預設運行於 `http://localhost:8000`。
您可透過 GET 請求 `http://localhost:8000/api/health` 來檢查健康狀態。

### 2. 準備上傳教材
請備妥您要轉換的教材檔案（例如 `material.pptx` 或 `document.pdf`）。

### 3. 發送 API 請求生成題庫
請使用 Postman 或 cURL 發送 POST 請求至 `/api/generate-question-bank`：

**使用 cURL 範例：**
```bash
curl -X POST http://localhost:8000/api/generate-question-bank \
  -F "materials=@material.pptx" \
  -F "question_types=true_false,single_choice" \
  -F "difficulty=medium" \
  -F "language=zh-TW"
```
**參數說明：**
- `materials` (檔案): 要解析的源檔案。
- `question_types` (字串): 想產生的題型，多種題型以逗號分隔 (可選參數：`true_false`, `single_choice`, `multiple_choice`, `fill_in_blank`, `essay`)。
- `difficulty` (字串): 難易度 (`easy`, `medium`, `hard`)。
- `language` (字串): 輸出語系，預設 `zh-TW`。

### 4. 取得產出檔案
API 將會回傳 JSON 格式的結果，包含生成的題目數量 (`questions_count`) 以及檔案的下載網址 (`download_url`)。
請直接透過瀏覽器或程式 (GET 請求) 訪問該 `download_url`，即可取得轉換完畢的旭聯格式 Excel 題庫。

## 注意事項
- 檔案上傳大小限制依系統 `config.py` 設定為主。
- 上傳的 MP4 影片若無內建或附帶語音/字幕，產出效果可能受限。
