# 開發環境建置指南

本文件旨在協助開發人員快速建置本機 Flask 開發環境，以及如何執行單元測試。

## 系統需求
- Python 3.8 或以上版本
- 針對讀取不同格式需要系統層級相依套件

## 環境建置步驟

### 1. 建立並啟動虛擬環境
在專案根目錄下開啟終端機執行：
```bash
# 建立虛擬環境
python -m venv venv

# 啟動虛擬環境 (macOS / Linux)
source venv/bin/activate

# 啟動虛擬環境 (Windows Command Prompt)
venv\Scripts\activate
```

### 2. 安裝相依套件
本專案依賴 Flask (API)、Pandas 與 OpenPyXL (Excel處理)，以及多種檔案資料解析套件（PyPDF2, python-pptx, python-docx 等）。
```bash
pip install -r requirements.txt
```

### 3. 環境變數設定
請複製專案內的 `.env.example` 並重新命名為 `.env`。
根據您的本機環境，修改配置檔內的參數設定（包含上傳路徑、限制大小等）。

## 啟動服務
於終端機執行以下腳本啟動 Flask 開發伺服器：
```bash
python scripts/start_server.py
# 或是直接使用 Make 指令
make run
```
伺服器預設將運行於 `http://localhost:8000`。

## 執行測試與代碼檢查
本專案已配置完整的自動化測試框架與排版除錯工具，可利用 `Makefile` 快速執行：
- **執行所有測試 (Pytest)**：`make test`
- **執行 API 專屬測試**：`make test-api`
- **代碼自動排版整理 (Black)**：`make format`
- **代碼靜態語法檢查 (Flake8)**：`make lint`
