# 黑盒子題庫產生器 - 快速參考 (Quick Reference)

## 📍 項目位置
```
/programming/python/blackbox-question-generator/
```

## 🎯 目的
將教材 (PDF、PPTX、DOCX、MP4) → 自動生成 → 旭聯題庫 Excel

## 📦 核心組件

| 組件 | 功能 | 位置 |
|------|------|------|
| Input Adapter | 多格式檔案轉換 | `src/input_adapter/` |
| Canonical Content | 內容標準化 | `src/canonical_content/` |
| Question Generator | 出題引擎 | `src/question_generator/` |
| XuLink Exporter | Excel 匯出 | `src/xulink_exporter/` |
| API | Flask 服務 | `api/main.py` |

## 🚀 快速啟動

```bash
# 進入項目目錄
cd blackbox-question-generator

# 安裝依賴
pip install -r requirements.txt

# 啟動服務
python scripts/start_server.py

# 測試 API
make test-api
```

## 📋 旭聯規格參考

### 題型代碼
- `[TrueFalse]` - 是非題
- `[SingleChoice]` - 單選題
- `[MultipleChoice]` - 複選題
- `[FillInBlank]` - 填充題
- `[Essay]` - 問答題

### 欄位說明
- #Qtype: 題型代碼
- Subject: 題目
- Option: 選項
- Weight: 分數
- Difficulty: 難度 (1-5)
- Answer: 答案標記 (y/n)

詳見: `test_rule.md`

## 💻 API 端點

```
GET  /api/health              - 檢查服務狀態
GET  /api/config              - 查看配置
POST /api/generate-question-bank  - 生成題庫
GET  /api/download/<filename> - 下載檔案
```

## 📝 使用流程

1. **準備教材** - 上傳 PDF/PPTX/DOCX/MP4
2. **指定參數** - 選擇題型、難度、語言
3. **生成題庫** - API 自動處理
4. **下載 Excel** - 取得旭聯規格的題庫

## 🤝 快速問題

**Q: 如何上傳多個檔案?**
A: 使用 API 的 multipart/form-data，同時上傳多個 `materials` 欄位

**Q: 支持哪些難度?**
A: easy (1), medium (3), hard (5)

**Q: 生成速度?**
A: 單個檔案通常 < 5 秒

**Q: 輸出文件名?**
A: `training_question_bank_YYYYMMDD_HHMMSS.xlsx`
