# 使用者操作手冊

## 系統需求
請確認您的電腦已安裝本程式或取得已打包好的執行檔 (.exe / .app)。
需要準備欲檢查的測驗題庫 Excel 檔案 (附檔名須為 `.xlsx` 或 `.xls`)。

## 操作步驟

### 1. 啟動程式
執行程式後，會先看到數秒鐘的「旭聯學習題庫檢驗程式」開場閃屏畫面（Splash Screen），接著會自動跳出檔案選擇視窗。

### 2. 選擇檔案
- 在彈出的視窗中，瀏覽並選取您準備好的測驗 Excel 檔案。
- **提示**：您可以按住 `Ctrl` (Windows) 或 `Command` (Mac) 一次選擇多個檔案進行批次檢查。

### 3. 等待系統檢查
系統將自動在背景掃描檔案，檢查項目包含：
- 必備欄位名稱與順序（A欄到J欄）是否完全正確。
- 題目難度是否在 1~5 之間。
- 整份考卷的配分總和是否達到 100 分。
- 是非、單選、複選題的「選項數量」與「正確答案(y)」規則是否符合。
- 填充與問答題是否有在第二行填寫參考解答。

### 4. 取得檢查結果
檢查完成後，系統會跳出提示視窗告知結果，並自動在您的原始檔案資料夾旁建立兩個新資料夾：
- **`output/` (過關檔案區)**：所有「完全沒有錯誤」且「滿100分」的題庫檔案，會被安全地複製到這裡。您可以直接拿這些檔案去系統上傳。
- **`failed/` (錯誤報告區)**：如果檔案內有任何格式錯誤或題目邏輯問題，系統會在此資料夾內生成一份 `Quiz_Validation_Report.xlsx`。您可打開該報告查看具體是哪一個檔案的哪一題出錯，修正後再重新檢查。

## 注意事項
- 請確保題庫的 "#QTYPE" 標記存在於第一欄 (A欄)，系統會以此作為讀取題目的起點（上方的文字會被視為註解自動忽略）。
- 若第一欄以 "$" 符號開頭，該列會被當作註解自動跳過。
