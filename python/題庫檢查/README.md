# 旭聯學習題庫檢驗程式 (Question Bank Checker)

## 簡介
本專案提供一個圖形化介面 (GUI) 工具，用於自動化檢查「旭聯 e-learning 平台」的測驗題庫 Excel 上傳檔。透過嚴格的欄位與邏輯校驗，防止格式錯誤的題庫被匯入系統，大幅節省人工檢查時間與系統報錯成本。

## 檔案結構
- `Check.py`: 核心主程式 (跨平台相容，具備 Tkinter GUI)
- `Check_wins.py` / `Check_MacOS.py`: 各作業系統專用的特化腳本
- `題庫檢查.py`: 早期的簡易檢驗邏輯腳本
- `使用者操作手冊.md`: 針對出題者/終端使用者的操作步驟說明
- `開發環境建置指南.md`: 針對開發人員的環境設定與打包教學

## 功能特色
- **批次處理**：支援一次選取多個 Excel 檔案進行檢查。
- **嚴格欄位檢查**：確保 A~J 欄的標題與順序完全符合黃金標準。
- **題型邏輯防呆**：支援是非題、單選、複選、填充、問答題的選項與答案數量驗證。
- **難度與配分檢查**：檢查難度是否落於 1~5，並驗證單份考卷總分是否達 100 分。
- **自動分類匯出**：無錯誤的檔案自動複製至 `output` 資料夾；有錯誤的檔案則產生彙整報告 `Quiz_Validation_Report.xlsx` 於 `failed` 資料夾。
